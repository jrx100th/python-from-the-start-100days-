About This Project

A Flask + Jinja2 + pure JavaScript raycasting experiment.

This is a fully playable DOOM-style first-person shooter built from scratch for the browser.

The engine uses classic DDA raycasting (the same technique used in Wolfenstein 3D and the original DOOM). No WebGL, no external game engines — just math, <canvas>, and a lot of love for retro games.

Everything is served dynamically through Flask routes and Jinja templates.

TECH USED
  • Python + Flask
  • Jinja2 Templating
  • HTML5 Canvas + JS
  • Web Audio API
  • VT323 font + Tailwind
FEATURES
  • Real-time raycasted 3D
  • Enemy AI + sprite scaling
  • Mouse look + pointer lock
  • Hitscan shooting
  • Full HUD + minimap
  • Retro sound effects
